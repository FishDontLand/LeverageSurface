To run the main program, pleasse follow the steps:
1. git clone the project to local repository
2. pip install requirements.txt
3. cd to the directory that contains main.py
4. run: python main.py --iv_file [path-to-file] --s0 [s0] --v0 [v0] --r_d [r_d] --r_f[r_f] --kappa[kappa] --lambdas=[4 numbers separated by comma] --max_xis=[4 numbers separated by comma] --max_rhos=[4 numbers separated by comma] --n_time_steps [n_t] --n _strike_steps [n_s] --n_sim_paths [n_sim] --output_folder [o]
